## Introduction 

Open Jam game built in Godot 3.2.

## Controls

|  Input (QWERTY)   |   Action    |
| ----------------- | ----------- |
| A                 | Dodge Left  |
| D                 | Dodge Right |
| W                 | Dodge Up    |
| Mouse             | Aim         |
| Left Mouse Button | Shoot       |

## Editing in Godot

- Download Godot 3.2
- Download or clone the repo
- Import into Godot
